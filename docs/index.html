Notion Accessibility Enhancer

Notionをスクリーンリーダーで快適に使用するためのChrome拡張機能

ダウンロード

主な機能

不適切なクリック要素の修正

スクリーンリーダーで不要に読み上げられる要素を最適化します

ARIAラベルの最適化

ブロック要素に適切なラベルを付与し、内容を分かりやすく伝えます

ナビゲーション改善

サイドバーなどのナビゲーション要素を適切にマークアップします

画像の代替テキスト

画像に適切な代替テキストを自動で設定します

インストール方法

  1. GitHubからリポジトリをダウンロードまたはクローン
  2. Chromeで chrome://extensions を開く
  3. デベロッパーモードを有効化
  4. 「パッケージ化されていない拡張機能を読み込む」をクリック
  5. ダウンロードしたフォルダを選択

使い方

インストール後、自動的にNotionページのアクセシビリティが改善されます：

  • クリック可能な要素の適切な処理
  • ブロック要素への分かりやすいラベル付け
  • ナビゲーション要素の改善
  • 画像の代替テキスト自動設定
  • フォーカス表示の改善
  • コントラスト比の向上
